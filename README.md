Multiple Sequence Analysis:
Required Software: diamond 2.0.15, ParaAT 2.0, MAFFT 7.310, KaKs_Calculator 2.0
Environmental Dependencies: Python, Perl
Operating Systems: Windows, Linux (Windows-only steps will be marked accordingly)
Procedures:
1.Prepare the peptide (pep) and coding sequence (cds) files of target genes. Save the names of species to be analyzed (i.e., the prefixes of pep files) into a text file named species_list.txt.
2.Run the script diyibu_pairs.py.
3.Run the script zhiling.py.
4.Run the script 2.8.new.draw.py.(An XLSX file titled ks should be prepared beforehand.)
5.Run the script buchong.py (Windows system only).
6.Run the script dnj2.py (Windows system only).
